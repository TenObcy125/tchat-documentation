<h1>📖 TChat Command Reference</h1>

<p>Here is a list of all available chat commands you can use in <strong>TChat</strong>. These commands allow you to manage your tasks, communicate more efficiently, and interact with the chat system.</p>

<h2>🛠️ Basic Commands</h2>
<ul>
  <li><code>/help</code> – Show this help menu with all available commands.</li>
  <li><code>/exit</code> – Disconnect from the chat and close the client.</li>
</ul>

<h2>📋 TODO Commands</h2>
<ul>
  <li><code>/todo-add</code> – Add a new task to the shared TODO list.</li>
  <li><code>/todo</code> – Show all current tasks in the shared TODO list.</li>
  <li><code>/todo-set &lt;task_id&gt  &lt;true / false&gt;</code> – Mark a task as completed by its ID.</li>
</ul>

<h2>📁 File Upload Commands</h2>
<ul>
  <li><code>/upload;</code> – Upload a file to the server for other users to download.</li>
</ul>

  <strong>Tip:</strong> You can always type <code>/help</code> in the chat to remind yourself of available commands.
